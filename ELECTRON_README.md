# Electron App Usage Instructions

1. Install dependencies:

   - Open a terminal in the RadarApp folder.
   - Run: `npm install`
   - Ensure Python is installed and available in your PATH.

2. Start the Electron app:

   - Run: `npm start`
   - This will launch the Flask backend and open the frontend in a desktop window.

3. The Electron window will display your web app, served from the Flask backend at http://localhost:5000.

4. To stop the app, simply close the Electron window. The backend will shut down automatically.

---

If you encounter issues with Python or Flask, make sure you have all required Python packages installed (e.g., Flask, flask_cors, requests, etc.).
